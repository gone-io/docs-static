    • Installing gonectr and mockgen
      • Creating a Web Project
        • Running the Project

        # Creating a Simple Web Project

        # Installing gonectr (opens new window) and mockgen (opens new window)

        go install github.com/gone-io/gonectr@latest
go install go.uber.org/mock/mockgen@latest

        # Creating a Web Project

        gonectr create myproject

        # Running the Project

        cd myproject
gonectr run ./cmd/server

        Or use run Make command if you have installed make (opens new window):

        cd myproject
make run

        Or with docker compose:

        cd myproject
docker compose build
docker compose up
